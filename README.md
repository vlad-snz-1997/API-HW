Данная программа позволяет вычислить IP адрес клиента и сохранить информацию о пользователе на Яндекс диск.
В работе реализованы 2 версии взаимодействия с API.
В первой версии представлена  непосредственная работа с яндекс диском, скрывая функции и классы для получения информации об IP.
Во второй версии реализовано взаимодействие со всеми классами и функциями в главной функции "main". Преимущество данной версии заключается в открытом доступе и возможности "по шаговой" работы с каждым классом для получения раздельной информации.
